# CoffeeShop

A modern Android application for a Coffee Shop, built using Kotlin and Firebase.

## Features

- **Category Browsing:** Browse through different categories of coffee and snacks.
- **Popular Items:** View the most popular items.
- **Special Offers:** Check out special deals and items.
- **Real-time Updates:** Powered by Firebase Realtime Database.
- **Modern UI:** Built with Material Design components and Glide for efficient image loading.

## Tech Stack

- **Language:** Kotlin
- **Architecture:** MVVM (Repository pattern)
- **Database:** Firebase Realtime Database
- **Image Loading:** Glide
- **UI Components:** RecyclerView, ConstraintLayout, ViewBinding

## Project Structure

- `Activity/`: Contains the main entry points of the app (Splash, Main).
- `Adapter/`: Custom adapters for displaying lists of categories and items.
- `Domain/`: Data models for Categories and Items.
- `Repository/`: Handles data operations and Firebase interactions.

## Setup

1. Clone the repository.
2. Add your `google-services.json` file to the `app/` directory.
3. Build and run the project using Android Studio.
